# Screenshots

Store portfolio-facing screenshots for the repository in this directory.

Recommended files:

- `dashboard-overview.png`
- `findings-tab.png`
- `diagnostics-runtime-events.png`
- `plugin-drilldown.png`
- `diagnostic-session.png`

Suggested capture guidelines:

- use a reasonably sized browser window
- keep sensitive site data out of the screenshot
- prefer realistic data over empty states where possible
- crop tightly enough to keep the UI readable in GitHub

Once screenshots are added, link them from the main `README.md` so the repository is easier to understand at a glance.
